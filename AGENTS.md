# Classics Club

## Vision
The Classics Club is a premium web app for discovering, tracking and discussing the world's greatest books and films.

It is not just a tracker. It should feel like a curated cultural journey for Chris and Brooke.

## Design principles
- Premium
- Calm
- Minimal
- Spoiler free
- Beautiful typography
- Fast

Inspirations:
- Letterboxd
- Apple TV
- Goodreads
- Notion

## Tech
- React
- Vite
- Local storage initially
- Open Library for books
- TMDB later for films

## Code principles
- Small reusable components
- One responsibility per component
- Keep pages under 200 lines where practical
- Avoid duplication
- Preserve existing functionality unless explicitly asked to change it
